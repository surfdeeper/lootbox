# Lootbox Game

Sci-fi themed lootbox/idle game built with React and TypeScript.

## Commands

### /bot
Run the bot testing system to test game mechanics and find bugs. The bot simulates gameplay by performing random actions (opening chests, selling items, upgrading, battling, etc.) and validates game state after each action.

After tests complete, provides:
- Test results summary (bugs found, actions performed)
- Analysis of final game state averages
- 5 suggested improvements based on gameplay analysis
